How to put a gif/image from internet in roblokis theme?
1: Open roblokis settings menu
2: Go to designer
3: Create a theme
- Input a name (must be more then 3 characters) 4: Click edit
5: Click the thing you want to make a gif/image
6: Where it says link, put The link of the gif/image (look below)

How to get gif/image link from internet?
1: Find the gif/image on the internet (ex. google)
2: Right click on it and click open image in new tab (optional)
3: Right click it then click "Copy image link".
How to use Pre-made Roblokis Themes?
1: Open roblokis settings menu
2: Go to designer
3: Create a theme
- Name and description are optional (will be set as the pre-made theme name)
4: Click import then select the .Roblokis File
5: Click Create then you should be able to select it.
